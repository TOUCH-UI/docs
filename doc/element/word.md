## 大小设置

```html
<span class="e_size-s">临</span>
```

支持：`e_size-xs` `e_size-s` `e_size-m` `e_size-l` `e_size-xl` `e_size-xxl` `e_size-xxxl` `e_size-xxxxl`

## 色彩设置

```html
<span class="e_red">文字</span>
```

## 加粗

<span class="e_red">文字</span>
```

## 上标

```
